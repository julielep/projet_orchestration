## Description

Mini API de calcul avec FastAPI, permettant :

- D’envoyer des opérations mathématiques via l’API.  
- De stocker les valeurs `a`, `b` et le `result` dans une base SQLite.  
- De récupérer toutes les opérations via un endpoint GET `/data/`.  
- D’utiliser une interface Streamlit pour saisir des opérations et afficher le résultat.
